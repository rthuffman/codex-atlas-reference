codex-atlas-reference

Public repository for certified historical Codex Atlas reference packs. The first v1 family is for historical U.S. officeholder Person and HoldsSeat records materialized with the reserved 72588 ProspectusID band.

Trust Boundary

Only operator-reviewed reference slices belong here. Raw Prospectus exports, reconciliation reports, uncertain identity matches, and other pre-certification artifacts stay in the private codex-workbench repository.

Bundle Shape

A release asset is a .tar.gz containing:

manifest.json
packs/
  us_holders_house/
    congress_001.json
    ...

manifest.json declares bundle_version, requires_seeds_bundle_version, requires_atlas_schema_version, release tier, corpus date, and the pack/file list. Each slice is a JSON document with records[]. Person rows use VertexType: "Person". Holder rows use EdgeType: "HoldsSeat", from_prospectus_id for the person 72588 ID, and to_prospectus_id for the structural office 72587 ID.

Validation

python tools/validate_reference_bundle.py .
python -m pytest tests -q
